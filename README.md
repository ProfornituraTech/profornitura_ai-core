# Profornitura AI – ERPNext Automation Platform

Questa è l'app ufficiale **Profornitura AI** per ERPNext/Frappe v15.

## Contenuti principali

- Modulo Gare (Doctype `Gara` + workflow M2)
- Timeline AI (Scadenza Gara, TimelineAI Event, Timeline Gara Evento)
- Moduli AI:
  - DocAI Report
  - DecisionAI Log
  - AI Request Log
- ANAC Regola CIG-CPV
- Logging:
  - Gara Log Operazioni
  - API Log
- Bozza Offerta

I dati demo inclusi permettono di vedere un ciclo completo di gara con:
- pubblicazione bando
- Q&A
- scadenza offerte
- invio offerta
- apertura buste
- aggiudicazione provvisoria/definitiva
